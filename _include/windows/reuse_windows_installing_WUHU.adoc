Las utilidades unificadas de host de Windows (WUHU) son un conjunto de programas de software con documentación que le permite conectar equipos host a discos virtuales (LUN) en UNA SAN de NetApp. NetApp recomienda descargar e instalar el kit de utilidades más reciente. Para obtener información e instrucciones sobre la configuración de WUHU, consulte link:https://docs.netapp.com/us-en/ontap-sanhost/hu_wuhu_71_rn.html["Documentación de Windows Unified Host Utilities"] Y seleccione el procedimiento de instalación para su versión de Windows Unified Host Utilities.
